# Información de Copyright para App Store Connect

## Información de Copyright

**Campo requerido en App Store Connect:**

```
Copyright © 2024 Pau Benet Prat
```

O si prefieres incluir "Todos los derechos reservados":

```
Copyright © 2024 Pau Benet Prat. Todos los derechos reservados.
```

## Formato Estándar

El formato de copyright para App Store Connect sigue este patrón:

- **Formato simple**: `Copyright © [Año] [Nombre del Propietario]`
- **Formato completo**: `Copyright © [Año] [Nombre del Propietario]. Todos los derechos reservados.`

## Notas Importantes

1. **Año**: Usa el año actual (2024) o el año en que se publica la app
2. **Símbolo ©**: Asegúrate de usar el símbolo de copyright (©) y no "(c)" o "Copyright"
3. **Nombre**: Usa el nombre legal del propietario de los derechos
   - Si es una persona: Nombre completo
   - Si es una empresa: Nombre legal de la empresa

## Dónde Configurarlo en App Store Connect

1. Ve a **App Store Connect**
2. Selecciona tu app
3. Ve a **App Information** (Información de la app)
4. Busca el campo **"Copyright"** o **"Derechos de autor"**
5. Introduce: `Copyright © 2024 Pau Benet Prat`

## Información de Derechos sobre el Contenido

Si App Store Connect también te pide información sobre los derechos del contenido, puedes usar:

**Opción 1 (Simple):**
```
Copyright © 2024 Pau Benet Prat
```

**Opción 2 (Completa):**
```
Copyright © 2024 Pau Benet Prat. Todos los derechos reservados.
```

**Opción 3 (Si es código abierto):**
```
Copyright © 2024 Pau Benet Prat. Disponible para fines educativos.
```
